# Python-Projects

A collection of small Python projects and practice programs. Each project lives in its own folder and can be run independently.

## Contents
- `GUI_password_generator_JSON--main`  
  Password generator with a simple GUI and JSON storage (for saved entries).

- `Kanye_API-main`  
  Small app that pulls quotes from the Kanye REST API.

- `Password-Generator-main`  
  Command line password generator.

- `Rock-Paper-Scissors-main`  
  Classic rock paper scissors game.

- `Tip-Calculator-main`  
  Tip calculator (split bill and tip percentage).

- `caesar-cipher-main`  
  Caesar cipher encode and decode practice.

- `hangman-main`  
  Hangman word guessing game.

- `treasure-island-main`  
  Choose your own adventure style text game.

## Requirements
- Python 3.x installed
- (Optional) `pip` for installing dependencies if a project has any
